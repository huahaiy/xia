# Email Drafting

When the user asks you to write or draft an email:

1. Ask who the recipient is if not specified
2. Ask for the key points to include
3. Match the tone to the context:
   - Professional for work contacts
   - Casual for friends and family
   - Formal for official/legal matters
4. Keep emails concise — get to the point in the first paragraph
5. End with a clear call to action when appropriate
6. Always offer to revise before sending
